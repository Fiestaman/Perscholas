  3. Clip-path property: takes a value to define the shape of the circle.

2. What is Z-index?

Z-index is a CSS property that specifies the stacking order of elements that overlap. An element with a greater z-index will always be in front of an element with a lower z-index.

3. What is a Vendor Prefix and What is it's Purpose?

A vendor prefix is a string that is added to the name of a CSS property or JavaScript function to indicate that it is a proprietary feature of a particular browser vendor. For example, the -webkit- prefix is used by Webkit browsers (such as Chrome and Safari), the -moz- prefix is used by Mozilla browsers (such as Firefox), and the -ms- prefix is used by Microsoft browsers (such as Internet Explorer).
Vendor prefixes are used because browsers often implement new features before they are standardized. By adding a vendor prefix to a feature, browser vendors can test the feature in their browsers and get feedback from users without worrying about breaking compatibility with other browsers.

4. What is the difference between CSS Grid and Flexbox?

CSS Grid is a two-dimensional layout system, which means that you can use it to create layouts that have both rows and columns. It is also very flexible, and you can use it to create complex layouts that would be difficult or impossible to create with Flexbox.
Flexbox is a one-dimensional layout system, which means that you can only use it to create layouts that have rows or columns, but not both. It is also not as flexible as CSS Grid, but it is easier to learn and use.

6. What is CRUD?

CRUD is an acronym for Create, Read, Update, and Delete. It is a term used in computer programming to describe the four basic operations that can be performed on data.

  • Create: This operation creates a new record of data.
  • Read: This operation retrieves a record of data from the database.
  • Update: This operation modifies an existing record of data.
  • Delete: This operation removes a record of data from the database.

7. What is OOP?

OOP stands for Object-Oriented Programming. It is a programming paradigm that is based on the concept of objects. Objects are entities that have state and behavior. The state of an object is represented by its data, and the behavior of an object is represented by its methods.

OOP is a powerful programming paradigm that has many advantages over other programming paradigms. Some of the advantages of OOP include:

  • Encapsulation: Encapsulation is the process of hiding the internal details of an object from the outside world. This makes it difficult for other objects to accidentally modify the state of an object.
  • Inheritance: Inheritance is the process of creating a new object that inherits the properties and methods of an existing object. This can save a lot of time and effort, as you do not have to write the code for the properties and methods of the new object from scratch.
  • Polymorphism: Polymorphism is the ability of an object to respond to different messages in different ways. This allows you to create objects that can be used in a variety of different ways.
